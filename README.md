# Water Intake Calculator & Advisor

A lightweight tool to calculate daily hydration needs and provide personalized health recommendations based on physical parameters and activity levels.

## Features
- Precise daily water goal calculation.
- Adjustments for physical activity and climate.
- Hourly intake distribution advice.
- Health tips and dehydration warnings.
